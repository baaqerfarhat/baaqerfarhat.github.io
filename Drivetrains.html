  • Mecanum Drivetrain Assembly

    This robot features a Mecanum wheel drivetrain for omnidirectional movement and a PID control algorithm for precise navigation.

  • Multi-Level Block Retrieval and Delivery Robot

    2021 - 2022

    This robot is equipped with a Mecanum wheel drivetrain, offering omnidirectional maneuverability. Integrated into its design is a precision-engineered linear actuator mounted on an advanced elevator mechanism. This configuration facilitates vertical articulation, enabling the robot to efficiently reach and operate at multiple elevation tiers for the targeted placement and retrieval of designated blocks.

  • Scissors Mechanism

    2019 - 2020

    This scissors mechanism was made with custom 3D-printed connectors affixed to the termini of aluminum extrusions. These extrusions are centrally connected with a bearing,enabling the mechanism to achieve a compact form factor in its fully retracted state.

  • Linear Actuator

    2019 - 2020

    Simple linear actuator developed by using a very simple gearbox to increase torque for lifting the robot.

  • Intelligent Fan Arduino Project

    2019 - 2020

    This project detects the temperature in a room, prints it on LCD screen, then operates the servo (acting as a fan) when the temperature is above the threshold provided.
